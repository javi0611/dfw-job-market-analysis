# DFW Data Analyst Job Market Dashboard

## Overview
An Excel-based analysis of 50 data analyst job postings in the Dallas-Fort Worth metropolitan area, designed to identify hiring trends, salary benchmarks, and the most in-demand skills for aspiring data analysts.

**Author:** Javier Villegas  
**Tools:** Microsoft Excel, Python (openpyxl for automation)  
**Data:** 50 job postings from major DFW employers (February 2026)

## Key Findings

### Skills in Highest Demand
- **SQL** appears in 100% of postings — it is non-negotiable
- **Excel** appears in 96% of postings
- **Python** appears in 72% of postings
- **Tableau** and **Power BI** are the top visualization tools

### Salary Landscape
- Entry-level roles range from $48K–$85K with an average midpoint around $64K
- Mid-level roles range from $65K–$100K with an average midpoint around $82K
- Finance and Defense industries pay the highest on average
- Government and Education roles pay the lowest

### Market Observations
- Healthcare is the largest hiring industry for analysts in DFW
- Over 60% of roles offer hybrid work arrangements
- The majority of postings require only a Bachelor's degree
- Entry-level roles outnumber mid-level roles nearly 2:1

| Sheet | Description |
|-------|-------------|
| **Job Postings Data** | Raw dataset of 50 postings with filters enabled |
| **Dashboard** | KPI summary cards with key metrics |
| **Skills Analysis** | Bar chart and table of most requested skills |
| **Industry Breakdown** | Pie chart and salary data by industry |
| **Salary Analysis** | Salary ranges by experience level and industry |

## How This Was Built
1. Collected job posting data from major DFW job boards
2. Structured data into a standardized format (title, company, salary, skills, etc.)
3. Built automated analysis using Python and openpyxl
4. Created charts and KPI dashboards directly in Excel

## Insights for Job Seekers
- **Must-have skills:** SQL + Excel (table stakes for any analyst role)
- **High-value additions:** Python and Tableau/Power BI significantly expand your options
- **Best entry points:** Healthcare and consulting firms have the most entry-level openings
- **Salary negotiation:** Entry-level candidates with Python skills can target the $65K+ range
